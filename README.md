Voici les composants principaux utilisés :

- VueX
- vueRouter
- Bootstrap
- Typescript
- Axios

- Améliorations Design

Je tiens à préciser que j'ai découvert VueJS à l'occasion de ce test. Je viens de ReactJS à la base.

```
npm install --legacy-peer-deps
```
